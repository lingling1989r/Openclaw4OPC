# OpenClaw 知识分享系列课程

> Building in Public - 公开学习，利他成长

---

## 📚 课程简介

**为什么要学习 OpenClaw？**

AI 是变革型新事物，这是一个行业共识。作为变革的共识，我们都要积极融入，不要做"头铁"的人。

**本系列课程帮助你：**
- 掌握 AI 工具使用方法
- 打造个人 AI 助理
- 提升工作效率
- 实现自动化运营

---

## 📖 课程大纲

### 第一章：认知与案例

**学习目标：** 了解 OpenClaw 能做什么，看别人怎么用

**内容包括：**
- 1.1 OpenClaw 是什么？
- 1.2 案例分享：别人怎么用 OpenClaw
- 1.3 你的行业可以怎么用？
- 1.4 工具的边界与可能性

**你将获得：**
- 对 OpenClaw 的基本认知
- 行业应用灵感
- 案例参考素材

---

### 第二章：每天一个 Skill

**学习目标：** 动手实践，掌握具体技能

**内容包括：**
- 2.1 Skill 是什么？
- 2.2 如何使用现有 Skill？
- 2.3 定制自己的 Skill
- 2.4 实战案例演练

**你将获得：**
- 动手操作经验
- 技能掌握能力
- 实战练习机会

---

### 第三章：需求梳理

**学习目标：** 学会清晰表达需求

**内容包括：**
- 3.1 为什么要梳理需求？
- 3.2 宏观行业需求分析
- 3.3 微观个人需求梳理
- 3.4 工作流盘点与优化

**你将获得：**
- 需求梳理方法
- 工作流优化思路
- 外包沟通能力

---

### 第四章：行动营 - 外包模式

**学习目标：** 实践出真知

**内容包括：**
- 4.1 外包模式介绍
- 4.2 AI 智能助理外包
- 4.3 实战项目演练
- 4.4 成果验收与反馈

**你将获得：**
- 实际项目经验
- 成果交付能力
- 快速见效体验

---

### 第五章：组织模式

**学习目标：** 规范化、规模化

**内容包括：**
- 5.1 从外包到自建
- 5.2 团队协作规范
- 5.3 自动化流程设计
- 5.4 效率提升方案

**你将获得：**
- 规模化思维
- 团队管理方法
- 系统化能力

---

### 第六章：获客与变现

**学习目标：** 实现商业闭环

**内容包括：**
- 6.1 海外获客策略
- 6.2 国内获客策略
- 6.3 三大业务拆解
- 6.4 变现路径规划

**你将获得：**
- 获客方法论
- 变现思路
- 商业模型
