## 什么是 Skill？（3分钟）

### 职场比喻

大家有没有想过这个问题：

> **一个新员工入职，公司会做什么？**

是不是要**培训**？

- 入职培训（规章制度）
- **技能培训**（Excel、PPT、工具）
- **业务培训**（产品知识、工作流程）

**没有培训的员工 = 没有 Skill 的 AI**

→ 只能打杂，专业的事干不了

---

### Skill = AI 的岗前培训

| 职场概念 | 对应 OpenClaw |
|---------|--------------|
| 技能培训 | Skill |
| 培训课件 | SKILL.md |
| 培训后能做的事 | Skill 能解决的问题 |

**核心价值：**
- ✅ 省钱：不用每次从头教
- ✅ 高效：培训一次，终身可用
- ✅ 专业：2小时学的，AI 2秒搞定
- ✅ 标准：像 SOP 一样稳定输出

## 如何使用？（2分钟）

### 命令示例

```bash
# 查看已安装技能
openclaw skills list

# 安装新技能
clawhub install weather

# 直接开聊
"北京今天天气怎么样？"
"帮我写个请假条"
```

## 技术结构（2分钟）

```
我的Skill/
```
