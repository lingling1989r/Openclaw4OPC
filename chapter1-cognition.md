## 1.1 OpenClaw 是什么？

> 一个可以帮你打造个人 AI 助理的工具

OpenClaw（俗称"大龙虾"）是一个可以在你自有设备上运行的个人 AI 助手。

### 核心能力

| 能力 | 说明 |
|------|------|
| 多渠道接入 | 飞书、微信、Telegram、Discord、Slack 等 |
| 本地运行 | 数据存在自己电脑上，隐私安全 |
| 技能系统 | 每天一个 SKILL，持续进化 |
| 多 Agent | 可以同时运行多个 AI"员工" |

## 1.2 认知框架：不做什么比做什么更重要

### 三个核心认知

| 认知 | 解释 | 行动 |
|------|------|------|
| **不做什么** | 避免踩坑，避免做回报低的事 | 别人没跑过的领域，先让子弹飞一会儿 |
| **做什么** | 做有复利的事 | 一次投入，长期复用 |
| **盘需求** | 知道自己要什么，比学工具更重要 | 先想清楚要解决什么问题 |

### 为什么要"让子弹飞"？

> **案例：** 有人问"微信能做吗？"
> 
> 如果你不是从事这一行的，别人没跑过的领域，先观望。
> 
> 市场水很深，不是每个想法都能落地。
> 
> **结论：** 盘需求，赢了一大半。

## 1.3 AI 的边界：理论边界 vs 实际边界

### 两种边界

| 边界类型 | 解释 | 示例 |
|----------|------|------|
| **理论边界** | 技术上 AI 能做什么 | 理论上可以用 AI 做任何事 |
| **实际边界** | 市场上别人验证过能做什么 | 别人真的跑通并拿到结果的 |

### 为什么实际边界更重要？

- **理论可行 ≠ 实际能做**
- **有人做过 ≠ 你能做**
- **做的人多 ≠ 适合你**

> **金句：** "别人没跑过的领域，先让子弹飞一会儿。"
> 
> **行动：** 先看案例，再决定要不要动手。

## 1.4 我们的边界

### 什么是"我们的边界"？

| 问题 | 答案 |
|------|------|
| 哪些事我想放权给 AI？ | 重复性劳动、信息整理、数据分析 |
| 哪些事我不想交给 AI？ | 核心决策、人情世故、需要创造力的事 |
| 我的核心能力是什么？ | 判断力、审美、战略思考 |

### 为什么要想"边界"？

1. **效率最大化** — 把重复的事交给 AI，专注高价值的事
2. **不踩坑** — 知道 AI 擅长什么、不擅长什么
3. **心理舒适** — 明确哪些事必须自己掌控

## 1.5 龙虾 vs 传统 APP：最大的差异

### 传统 APP

| 特点 | 说明 |
|------|------|
| 功能固定 | 做什么是开发好的 |
| 门槛固定 | 会用就会用，不会用还是不会 |
| 迭代慢 | 功能更新需要开发周期 |

### 龙虾（OpenClaw）

| 特点 | 说明 |
|------|------|
| **可定制** | 你让它做什么，它就做什么 |
| **可学习** | 每天一个 SKILL，持续进化 |
| **可成长** | 越用越懂你，越用越顺手 |

### 最大的差异

> **传统 APP：** 你适应工具  
> **龙虾：** 工具适应你

## 1.6 畏难情绪：大家在怕什么？

### 常见畏难点

| 畏难点 | 真相 |
|--------|------|
| "不懂技术，能用吗？" | 傅盛的案例里，初中生都能用 |
| "部署很复杂吧？" | 有一键部署，50 块就能跑起来 |
| "我能用它做什么？" | 先盘需求，再找答案 |
| "别人都那么厉害，我不行" | 大家都从 0 开始 |

### 如何克服？

1. **看案例** — 别人能做到，你也能
2. **从小开始** — 先跑通一个简单用例
3. **持续进化** — 每天一个 SKILL，慢慢就厉害了

> **金句：** "工具怎么用，比工具本身更重要。"

## 1.7 案例拆解：傅盛"龙虾养成日记"

> 这是一个值得深度拆解的案例，因为它展示了：
> 1. AI 怎么用（明线）
> 2. 怎么让更多人知道（暗线）

### 明线：龙虾怎么用

| 故事 | 我们能学什么 |
|------|--------------|
| 8个AI"员工"同时工作 | AI 可以同时分担多种工作 |
| 48小时139技能/50块 | 低成本快速验证新技能 |
| 健康管家监督康复 | 个人知识管理/提醒助手 |
| 交易官 7×24小时分析 | 自动化数据分析 |
| 初中生都能用 | 门槛低，重在动手 |

### 暗线：营销手法拆解

**先问自己：这份 PPT 各大社群都在传，最重要的是什么？**

| 暗线手法 | 目的 | 我们能借鉴什么 |
|----------|------|---------------|
| 讲故事 > 讲功能 | 真实故事更有说服力 | 用案例代替功能表 |
| 现场演示 > 列参数 | 眼见为实 | 现场演示更有冲击力 |
| "50块139技能" | 低成本吸引力 | 强调投入产出比 |
| 引导加群 | 私域沉淀 | 引流到持续触达的地方 |
| 初中生都能用 | 降低心理障碍 | 告诉用户"门槛很低" |
